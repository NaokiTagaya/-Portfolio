# フジノキ

フジノキはポテパンキャンプ様にてカリキュラムを学んだ後、ポートフォリオとして作成したものです。

# 特徴

フジノキはRuby On Railsをベースに、下記Gemを使用しております。
  ・Vue.js
  ・JQuery
  ・bootstrap
  ・carrierwave
  ・devise
  ・geocoder

フジノキでは、ユーザーがテイクアウトの店舗をサイトに登録し、他のユーザに共有することができます。
また登録された店舗に対してレビューを付けることができたり、気に入った店舗があればお気に入りに追加することができます。
